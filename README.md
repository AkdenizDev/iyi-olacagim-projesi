## 📌 Proje Hakkında

Selamlar! Yaklaşık 2 haftalık yoğun bir çalışma süreciyle geliştirdiğimiz bu projeyi sizlere sunmaktan mutluluk duyuyoruz. Bu projeyi, Türkiye'deki deprem gerçeğine karşı her bireyin bilinçlenmesi ve dilediği gibi düzenleyip geliştirebilmesi için açık kaynaklı olarak hazırladık.

> **"Umarız ki bu proje az da olsa size bir şeyler öğretir veya yardımcı olur. Hepiniz çok değerli insanlarsınız. Bunu asla ama asla unutmayın lütfen."** 🤍

## ✨ Özellikler

Proje, deprem anında ve sonrasında ihtiyaç duyulabilecek kritik bilgileri tek bir çatı altında toplar:

* 🌐 **Anlık Deprem Takibi:** Kandilli Rasathanesi verileri kullanılarak Türkiye'deki son depremlerin anlık listelenmesi.
* 🎥 **Eğitici İçerikler:** AFAD tarafından hazırlanan deprem bilgilendirme videoları ve metinleri.
* 📞 **Acil Durum Bilgileri:** 81 ilin tamamı için AFAD merkez konumları ve telefon adreslerine hızlı erişim.
* 🛠 **Geliştirilebilir Yapı:** Herkesin kendi ihtiyacına göre düzenleyebileceği esnek kod yapısı.

## 🚀 Kurulum

Projeyi kendi yerel ortamınızda çalıştırmak için aşağıdaki adımları takip edebilirsiniz:

1.  **Depoyu Klonlayın:**
    ```bash
    git clone https://github.com/carrotistekanka/iyi-olacagim-projesi.git
    ```

2.  **Proje Klasörüne Girin:**
    ```bash
    cd iyi-olacagim-projesi
    ```

3.  **Gerekli Bağımlılıkları Yükleyin:**
    ```bash
    npm install
    ```

4.  **Uygulamayı Başlatın:**
    ```bash
    npm start
    ```
    *(Veya projeniz hangi komutla çalışmasını ayarladıysanız, örneğin: `node app.js`)*
