## ✨ Özellikler

Projemiz deprem bilincini artırmak amacıyla aşağıdaki temel işlevleri sunmaktadır:

* 🌍 **Anlık Veri:** Kandilli Rasathanesi'nden çekilen anlık deprem bilgileri.
* 📚 **Bilinçlenme:** AFAD deprem bilgilendirme videoları ve metinleri.
* 📍 **İletişim & Konum:** 81 ilin AFAD kurumlarına ait güncel konum ve telefon adresi bilgileri.
